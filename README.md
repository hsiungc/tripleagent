# TripleAgent

A Unified Capability, Behavioral Safety, and Harmful-Compliance Evaluation Pipeline for LLM Agents



Begin by creating and activating a virtual environment and adding API keys based on the agent being evaluated.

Run the following command to create the build system and install required packages:

```pip install -e .```


To run experiments:
```python -m tripleagent.run_experiment configs/experiments/default.yaml```
